◆ Start()

BOOL CThread::Start ( )

start the execution of the thread

Could not be started from itself, throws exception in this case. If thread is already started do nothing and returns TRUE. Close thread handle if needed. Change thread status to ThreadStateWaiting before starting and to ThreadStateFault if thread was not created sucsessfully

Returns
BOOl. Either thread was sucsessfully created or not
